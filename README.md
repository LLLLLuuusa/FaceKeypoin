# 脸部关键点识别


### 项目描述

​	练手项目,基于Tensorflow脸部98关键点识别,可以识别出脸部的98个关键点,是人脸识别,表情识别,人脸密码等诸多功能的底层实现


### 使用
#### 使用前请确保具有NVIDIA GPU 和 Tensorflow环境
`pip install tensorflow-gpu==2.5.0`
`pip install opencv`


### 运行
`python inference.py`

### 训练
`python train.py --epocho=300 --train_path=/data/xxx`
